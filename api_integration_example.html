Available API Endpoints

1. Search Documents

POST /api/search
{
  "query": "search terms",
  "page": 1,
  "per_page": 20,
  "file_types": ["Word Document", "PDF Document", "Excel Spreadsheet"]
}

2. Get Statistics

GET /api/stats

Returns total documents, databases, and last update time.

3. Get Popular Words

GET /api/top-words

Returns the top 10 most common words across all documents.

4. Open File

POST /api/file/open
{ "file_path": "C:\\path\\to\\file.docx" }

5. Open Folder

POST /api/file/open-folder
{ "file_path": "C:\\path\\to\\file.docx" }
