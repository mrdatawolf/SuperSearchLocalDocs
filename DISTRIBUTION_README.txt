Thank you for using SuperSearch Local Docs!

QUICK START:
============

1. Run "DocumentIndexer.exe" to configure and index your documents
2. Run "DocumentSearch.exe" to start the search server
3. Open your web browser to: http://localhost:9000
4. Start searching!


WHAT'S INCLUDED:
================

DocumentIndexer.exe - Use this to set up and index your document collection
DocumentSearch.exe  - Use this to search through your indexed documents


DETAILED INSTRUCTIONS:
======================

Please read the "END_USER_README.md" file for complete instructions:
- First-time setup
- How to search
- Advanced features
- Multi-user setup
- Troubleshooting


SYSTEM REQUIREMENTS:
====================

- Windows 7 or later (64-bit)
- 2 GB RAM minimum
- Modern web browser (Chrome, Firefox, Edge)
- No Python installation required!


SUPPORT:
========

For help and troubleshooting, see:
- END_USER_README.md (included with distribution)
- Contact your system administrator


PRIVACY:
========

This software runs entirely on your local computer/network.
No data is sent to the internet.
All searches are performed locally.
